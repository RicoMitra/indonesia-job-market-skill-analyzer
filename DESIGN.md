# Design

## Aesthetic direction

Professional job-board analytics: warm paper neutrals, ink typography, restrained forest-green evidence accents, and compact data-first rows inspired by—not copied from—Wellfound.

## Dials

- Design variance: 4/10
- Motion intensity: 2/10
- Visual density: 6/10

## Tokens

- Background: `#f7f6f0`
- Ink: `#17201c`
- Muted text: `#64706a`
- Sidebar: `#eef1e9`
- Accent: `#237a57`

## Rules

- English UI; direct, evidence-based copy.
- Use native Streamlit elements, real labels, visible keyboard focus, and chart tables.
- No emoji UI, no gradients, no fake live data, no generic cards nested inside cards.
- Prefer subtle transitions only; respect reduced motion.
